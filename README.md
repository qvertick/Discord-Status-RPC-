## 🚀 Features

* **Абсолютная кастомизация:** Ставь любой текст в `details` и `state`.
* **Кликабельные кнопки:** Добавляй кнопки со ссылками на свой GitHub, портфолио или плейлист.
* **Кастомные иконки:** Поддержка больших и маленьких изображений с всплывающими подсказками.
* **Живой таймер:** Показывай, сколько времени уже запущена твоя «активность».

---

## 🛠️ Tech Stack

Проект построен на базе современного и легковесного стека для работы с Discord API:
* **Language:** Python 3.10+
* **Library:** `pypresence` (клиент Discord RPC на асинхронном Python)
